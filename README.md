# PaymenterRemoveBranding
RemoveBranding is a lightweight Paymenter extension that automatically removes the "Powered by Paymenter" branding from the footer of your website. This extension provides a clean way to hide the default Paymenter attribution link without modifying core files.



Automatically removes the "Powered by Paymenter" link and logo from all pages
Works seamlessly with all themes
No configuration required - simply enable and it works
